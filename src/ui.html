How to use

  1. Select any text layer
  2. Start typing Arabic in the textarea
  3. BONUS: select the layer before openning the plugin so you can edit the text.

Notes

  • If you forgot to select a layer and already started typing, just select the layer and click fix, then continue typing.
  • If the glyphs do not appear correctly, play around with the available options.
